Education K-12 / Assessments NAEP (National Assessment of Educational Progress) 小型 小型 ≈50 GB 开放访问 Web https://nces.ed.gov/nationsreportcard/ ⭐⭐⭐ 高 美国教育评估“全国成绩单” 考试成绩/问卷权重 教育质量、成就差距
Education HigherEd / Outcomes College Scorecard (US Dept. of Education) 小型 小型 ≈50–100 GB 开放访问 Web / API https://collegescorecard.ed.gov/data/ ⭐⭐⭐ 高 高校成本/债务/收入/毕业率等 表格/院校-项目级 绩效评估、择校分析
Education School System Common Core of Data (CCD) – NCES 小型 小型 ≈50–100 GB 开放访问 Web / API https://nces.ed.gov/ccd/ ⭐⭐⭐ 高 美国公立学校/学区基础数据 表格/时序 教育资源配置、学区研究
Education International Assessments OECD PISA 小型 小型 ≈50–100 GB 开放访问 Web / API https://www.oecd.org/pisa/data/ ⭐⭐⭐ 高 国际学生评估项目（阅读/数学/科学） 微观应答/权重/背景问卷 跨国比较、教育政策 可能需注册/条款
Education International Assessments IEA TIMSS & PIRLS (public microdata) 小型 小型 ≈50–100 GB 注册访问 Web https://www.iea.nl/data-tools/repository ⭐⭐⭐ 高 数学与科学/阅读素养评估（微观数据） 微观应答/题本/权重 教育测量/国际比较 需申请/协议
Education MOOC / Learning Analytics OULAD (Open University Learning Analytics Dataset) 小型 小型 ≈50 GB 开放访问 Web https://analyse.kmi.open.ac.uk/open_dataset ⭐⭐ 中 在线课程学习行为/成绩/资源 事件日志/表格 学习分析、辍学预测
Education MOOC / Clickstream HarvardX-MITx on edX (Dataverse) 小型 小型 ≈100–200 GB 开放访问 Dataverse https://dataverse.harvard.edu/dataverse/harvardx ⭐⭐⭐ 高 大规模 MOOC 点击日志与成果 点击/视频交互/讨论 学习行为建模 各批次数据独立
