Healthcare Cancer Genomics NCI Genomic Data Commons (GDC) 中型 中型 ≈10–50 TB 部分受控 Web / API https://portal.gdc.cancer.gov/ ⭐⭐⭐⭐ 极高 肿瘤多组学与临床配套平台（含 TCGA/TARGET 等） 测序/变异/表达/临床 受控数据需 dbGaP 授权
Healthcare Cancer Registry SEER Program (Surveillance, Epidemiology, and End Results) 小型 小型 ≈100–500 GB 注册访问 Web https://seer.cancer.gov/data/ ⭐⭐⭐⭐ 极高 美国癌症发病/生存登记项目 表格/微观记录 需注册/同意使用协议
Healthcare Clinical Trials ClinicalTrials.gov 小型 小型 ≈500 GB 开放访问 Web / API https://clinicaltrials.gov/ ⭐⭐⭐⭐ 极高 全球临床试验注册与结果披露 结构化注册表/文本 证据合成、药物再定位
Healthcare Critical Care EHR MIMIC-III / MIMIC-IV（PhysioNet） 小型 小型 ≈2 TB 受控访问 Web / PhysioNet https://physionet.org/ ⭐⭐⭐⭐ 极高 ICU 电子病历与监护数据 结构化EHR/时序/文本/波形 需 CITI 培训与数据使用协议
Healthcare ICU/EHR eICU Collaborative Research Database（PhysioNet） 小型 小型 ≈1 TB 受控访问 Web / PhysioNet https://physionet.org/content/eicu-crd/ ⭐⭐⭐ 高 多中心 ICU 数据库 EHR/护理/用药/时序 需培训/协议
Healthcare Public Health CDC Data Portal（含 WONDER 等） 小型 小型 ≈数十TB（分散） 开放访问 Web / API https://data.cdc.gov/ ⭐⭐⭐ 高 公共卫生监测与指标库 表格/时间序列 流行病学/政策评估
Healthcare Global Health WHO Global Health Observatory (GHO) 小型 小型 ≈100 GB 开放访问 Web / API https://www.who.int/data/gho ⭐⭐⭐ 高 全球健康指标与元数据 表格/时间序列 国际比较/SDG 监测
Healthcare Global Health IHME – Global Health Data Exchange (GHDx) 小型 小型 ≈数 TB（分散） 开放访问 Web https://ghdx.healthdata.org/ ⭐⭐⭐ 高 全球疾病负担 (GBD) 等项目数据入口 表格/模型输出 健康负担/风险因子
Healthcare Open FDA openFDA Datasets & API 小型 小型 ≈100–300 GB 开放访问 Web / API https://open.fda.gov/data/ ⭐⭐⭐ 高 不良事件、药械食品相关监管数据 结构化/文本 药品安全/信号检测
Healthcare COVID-19 Johns Hopkins CSSE COVID-19 (archival) 小型 小型 ≈10–50 GB 开放访问 GitHub https://github.com/CSSEGISandData/COVID-19 ⭐⭐⭐ 高 全球新冠疫情时空数据（历史） 表格/时间序列/地理 疫情分析/建模
Healthcare Imaging The Cancer Imaging Archive (TCIA) 中型 中型 ≈20 TB 部分受控 Web https://www.cancerimagingarchive.net/ ⭐⭐⭐ 高 多癌种医学影像共享平台 DICOM/影像/标注 项目分级别下载/使用条款
Healthcare EHR Synthetic Synthea (Synthetic Patient Generator) 小型 小型 ≈10–50 GB（可自生） 开放访问 Web / GitHub https://synthea.mitre.org/ ⭐⭐⭐ 高 基于规则与统计的合成电子病历 EHR/就诊/药物/病程 隐私友好、用于方法开发
